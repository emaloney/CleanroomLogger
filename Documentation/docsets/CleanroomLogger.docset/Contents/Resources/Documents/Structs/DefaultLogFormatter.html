DefaultLogFormatter

public struct DefaultLogFormatter: LogFormatter

The DefaultLogFormatter is a basic implementation of the LogFormatter protocol.

This implementation is used by default if no other log formatters are specified.

  • includeTimestamp

    If true, the receiver will include the timestamp of the log entry when formatting the log message.

    Swift

    public let includeTimestamp: Bool
  • includeThreadID

    If true, the receiver will include an identifier for the calling thread when formatting the log message. This makes it possible to distinguish between distinct paths of execution when analyzing logs generated by multi-threaded applications.
